\# 🛡️ Sovereign Code Auditor



\*\*Sovereign Code Auditor\*\* is a specialized SLM (Small Language Model) agent designed for high-security environments like Fintech and private dev shops. It scans Python and Ruby repositories for critical vulnerabilities without exposing full proprietary logic to the cloud.



\## 🚀 High-Value Features

\- \*\*Logic-Based Auditing:\*\* Uses Mistral's reasoning to find SQL Injections, Command Injections, and Logic Flaws that static tools miss.

\- \*\*Privacy-Centric:\*\* Designed to process code snippets locally and send only sanitized data for analysis.

\- \*\*Security-First Prompting:\*\* Provides CVSS-style risk scoring, Proof of Concepts (PoC), and remediation code.

\- \*\*Zero Training:\*\* Your code is never used to train external models.



\## 🛠️ Tech Stack

\- \*\*AI Engine:\*\* Mistral AI (Mistral-Small / Mistral-Large)

\- \*\*Frontend:\*\* Streamlit

\- \*\*Environment:\*\* Python 3.10+

\- \*\*Security:\*\* Dotenv for API key isolation
